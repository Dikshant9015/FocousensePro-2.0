Technology Stack

The Tools Behind The Project

Focusense Pro combines Python-based development with computer vision and AI libraries to build a practical, interactive, and easy-to-present system.

Py

Python

Python works as the main programming language because it is flexible, readable, and widely used in AI and machine learning projects.

St

Streamlit

Streamlit helps create an easy interface for presenting the project output in a fast and user-friendly way.

OC

OpenCV

OpenCV is used for webcam handling, frame processing, and computer vision operations that support gesture and image-based analysis.

DF

DeepFace

DeepFace adds emotion recognition capabilities by interpreting facial expression data from captured frames.

CZ

cvzone

cvzone simplifies hand tracking and visual utilities, helping the project build gesture-based analysis more smoothly.

AI

Integrated Intelligence

These technologies work together to transform raw user behavior into a meaningful productivity intelligence system.
